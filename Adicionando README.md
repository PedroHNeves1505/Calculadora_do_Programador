#  Calculadora do Programador

Uma ferramenta desenvolvida em **Python** para conversão entre diferentes sistemas de numeração.

---

## Funcionalidades

O projeto realiza a conversão bidirecional entre as seguintes bases:
* **Binário** (Base 2)
* **Quaternário** (Base 4)
* **Octal** (Base 8)
* **Decimal** (Base 10)
* **Hexadecimal** (Base 16)

## Tecnologias Utilizadas

* [Python 3.x](https://www.python.org/)
* Lógica de divisões sucessivas e potências posicionais.
* Manipulação de strings e tratamento de caracteres hexadecimais (A-F).

## Como Rodar o Projeto

1. Certifique-se de ter o Python instalado.

2. Clone o repositório:
git clone [https://github.com/pedrohneves1505/Calculadora_do_Programador.git](https://github.com/pedrohneves1505/Calculadora_do_Programador.git)

3. Acede à pasta do projeto:
cd Calculadora_do_Programador

4. Execute o script principal:
python app.py
